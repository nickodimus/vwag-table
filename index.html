Editing the selected token. Click empty space to deselect.

Splash Screen

When enabled, the player display shows the splash image instead of the map.

Grid

View

Rotate map (carries fog, tokens & stairs)

Drag an image onto the map to drop it. In Move mode, click an image or note to select it, drag to move, and press Delete to remove. Double-click a note to edit it. Notes are GM-only.

Player View

Rotate player view only

For physical play: zoom the player view until one grid square matches your minis on the TV, then lock. On each new map, calibrate its grid (drag a square) and the player zoom auto-adjusts so the square stays that exact size. Remembered between sessions.

Floors

Click any floor in the top-right list to jump to it. Pin the table to roam other floors without moving the players' screen, then Push to table when you're ready to show them.

Map library

Export saves every map to a JSON file you can back up or move to another machine.

Drag to move the map. Wheel to zoom. Alt+click to ping.

Place light

Small = torch, large = firepit. Right-click a light to remove it. Turn on Darkness (under Line of sight) so lights gate what players see.

Token

Click to drop, drag to move, right-click to remove. In Move mode, click a token to select it, then use the arrow keys to nudge it cell by cell. Loading an image re-skins the selected token, or sets the art for new tokens.

Area of Effect

Hover over the map to preview. Mouse wheel rotates the cone.

Measure

Measure one square

Use this when the map already has a grid and the overlay is off.
